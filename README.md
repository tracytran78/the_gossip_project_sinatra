# Bienvenue sur le projet
## Installation et utilisation
1. Placez vous où vous souhaitez cloner le 'projet'
2. ```$ git clone https://github.com/DevPomeis/projet ```
3. ```$ cd projet```
4. ```$ bundle install```
5. ```$ ruby app.rb```
6. Laissez vous guider par les instructions!
